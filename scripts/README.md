# Development Scripts & Tools

**Status:** Active

This directory contains utility scripts for environment setup, command logging, and dependency installation.

## Available Scripts

- `setup.sh` — Initialize environment and create necessary directories
- `command-log.sh` — Log commands with timestamps to command history
- `install-deps.sh` — Install project dependencies

## Usage

Run any script with bash:

```bash
bash scripts/setup.sh
bash scripts/command-log.sh "your message here"
bash scripts/install-deps.sh
```
