# CHARACTERS — Registry
## UNEXUSI Control Center

Characters are the named entities, personas, and aspects that operate within the UNEXUSI ecosystem. They are not decorative — each character has a defined role, a home in the system, and a functional relationship to the work.

---

## Character Types

| Type | Description |
|------|-------------|
| **Entity** | Deep architectural characters — define structure, flow, or domain |
| **Terminal Aspect** | The six operational aspects of the terminal consortium |
| **Fractal Cast** | Characters emerging from the MANDELBROT / fractal work (in development) |
